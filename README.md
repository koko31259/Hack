# Hack🛡️ InsureIQ — AI Insurance Intelligence Platform

Making Insurance Simple, Visual & Honest for Every Indian

🚀 AI-powered platform to analyze, compare & understand insurance policies using real-time AI.

🎯 About

InsureIQ is a web-based AI platform built for Finvasia Hackathon 2026 (Track: Insurance).

👉 It simplifies complex insurance policies using Claude AI
👉 Converts 40+ page documents into clear insights, visuals & recommendations

💡 78% of users don’t understand their policy — InsureIQ fixes that.

🚨 Problem
📜 Complex legal jargon
🙈 Hidden exclusions
🔇 No visual understanding tools
✅ Solution

AI + Visualizations + Simulations + Expert Chatbots

✨ Features
🧠 AI Policy Analyzer
Paste policy → get:
Score (0–100)
Coverage % breakdown
Risk alerts & AI suggestions
Visual charts (donut + bars)
🚨 Exclusion Highlighter
🟢 Covered
🔴 Excluded
🟡 Conditional
Risk score & summary
🔮 Scenario Simulator
“What if X happens?”
Outputs:
Claim payout
Out-of-pocket cost
Claim steps

📁 Claims Assistant
Step-by-step claim help

📖 Glossary
20+ terms in simple language
Search + real examples
🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript (SPA)
AI: Claude API (Anthropic)
Charts: Canvas API

👉 No frameworks • No dependencies • Fast & portable

🚀 Getting Started
git clone https://github.com/neurosparks/insureiq.git
cd insureiq
open index.html
For AI features:

Add API key in jsmain.js:

"x-api-key": "YOUR_API_KEY"

index.html   → All pages (SPA)
